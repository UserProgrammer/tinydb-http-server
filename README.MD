# Overview

An instance of TinyDB (https://pypi.org/project/tinydb/) with an HTTP frontend (https://realpython.com/python-http-server/).

This setup is only meant for local development/testing/sandboxing

## Setup

Before executing `script/build` and `script/server`, be sure to run `python -m venv venv` and `source venv/bin/activate`
to setup a Python virtual environment.
